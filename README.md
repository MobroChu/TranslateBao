TranslateBao
============

- TranslateBao(翻译助手)是一个Chrome浏览器扩展程序，支持Chrome浏览器网页画词翻译，支持面板翻译等。
- 目前正在开发中，致力于开发最佳体验的翻译应用。欢迎大家提出意见和建议。

======

### 开发版试用

> 在Chrome浏览器（或其他Chrome内核浏览器）中打开扩展程序，开启开发者模式，加载正在开发的扩展程序选择TranslateBao目录
